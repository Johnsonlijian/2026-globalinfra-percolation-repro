# R198 public release notes

Date: 2026-07-01

## Release identity

This release publishes the public reproducibility package for the R198 working
manuscript:

**Why road networks leave random-graph percolation: testing a local
embedded-merging constraint**.

The release is a code/data/figure archive, not a publication of the active
manuscript text.

## What changed from the earlier release candidate

- Updated public metadata from the earlier CSF-era title to the R198
  mechanism-first manuscript title.
- Added the R198 six-main-figure package under `figures/R198_final_main/` as
  SVG, PDF and PNG assets.
- Added `figures/R198_final_main/Figure_Source_Map_R198.csv` linking each R198
  main figure to its source evidence role.
- Updated `CITATION.cff`, `.zenodo.json` and `run_manifest.csv` to version
  `v1.0.1-r198`.

## Zenodo trigger note

`v1.0.1-r198` is a metadata-synchronized release created after Zenodo GitHub
archiving was enabled for this repository. Scientific content is unchanged from
the R198 public package; the version bump exists to emit a fresh GitHub release
event for DOI archiving.

## Public/private boundary

Included:

- code;
- derived source tables;
- public figure assets;
- data-source registry and runbook;
- citation and Zenodo metadata.

Excluded:

- active manuscript text;
- cover letters and reviewer-response drafts;
- internal rounds/logs;
- raw third-party OSM/Geofabrik/GHSL/WDI archives;
- credentials, private author/funding files and sync-conflict files.

## Claim boundary

The public evidence supports a bounded road-network model-transfer
decomposition. It does not claim a universal infrastructure percolation law,
a planarity-only causal theorem, full surrogate-chain mixing, or cross-domain
shared mechanism from water/rail/power checks.
